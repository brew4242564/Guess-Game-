# Guess Game!
A simple guessing game made with Vanilla Javascript.

## How it works!
The computer selects a random number between 1 and 6, and you have to guess it.
- If you guess the number get 2 points.
- If your number is +1 or -1 get 1 point.
- Otherwise you get 0 points.

## Objective of this project
Learn promises and async/await.
